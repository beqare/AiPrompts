## Website verbessern

```
Bitte analysiere den folgenden Code auf Optimierungspotenziale und verbessere ihn in allen relevanten Aspekten: Performance, Lesbarkeit, Modularität, Skalierbarkeit und Einhaltung moderner Best Practices für die entsprechende Programmiersprache oder Technologie. Erhöhe die Lesbarkeit, indem du klare Kommentare hinzufügst, sprechende Namen für Variablen, Funktionen und Klassen verwendest und die Struktur optimierst. Nutze semantisches HTML, moderne CSS-Techniken und effiziente JavaScript-Strukturen, falls relevant. Überprüfe, ob Frameworks oder Bibliotheken optimal integriert sind, und schlage sinnvolle Alternativen vor, falls nötig. Achte darauf, dass der Code responsiv und barrierefrei (Accessibility) ist.
Identifiziere und behebe potenzielle Fehler, Sicherheitslücken oder Schwächen. Beschreibe jede Änderung oder Verbesserung detailliert, dokumentiere größere Anpassungen mit kurzen Kommentaren und erkläre den Nutzen jeder Optimierung. Stelle sicher, dass der überarbeitete Code modernen Standards entspricht und optimal für zukünftige Entwicklungen geeignet ist.
```

## Website design

```
Bitte entwerfe ein neues Design für diese Website, das modern, benutzerfreundlich und visuell ansprechend ist. Verwende dabei ausschließlich einen Darkmode als Basisdesign. Die primäre Akzentfarbe soll #ea97ff sein, und sie sollte elegant in Buttons, Links, Hover-Effekte und andere interaktive Elemente integriert werden.
Sorge dafür, dass die Farbpalette harmonisch ist, mit gutem Kontrastverhältnis für Texte und Elemente, um eine optimale Lesbarkeit zu gewährleisten. Nutze moderne Design-Prinzipien wie klare Typografie, konsistente Abstände und ansprechende visuelle Hierarchien.
Achte auf eine durchdachte Benutzerführung (UI/UX), die intuitive Navigation und klar strukturierte Inhalte ermöglicht. Beschreibe das Design detailliert, inklusive Schriftarten, Abständen, Layouts und Effekten. Falls möglich, füge Vorschläge für ansprechende Animationen oder Übergänge hinzu, die das Design lebendiger machen. Halte das neue Design responsiv, damit es auf allen Gerätetypen, einschließlich Smartphones und Tablets, optimal funktioniert. Stelle sicher, dass Barrierefreiheit (Accessibility) berücksichtigt wird, und erkläre abschließend die Vorteile der Designänderungen.
```

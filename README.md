# AiPrompts

## Website verbessern

```
Bitte analysiere den folgenden Code auf Optimierungspotenziale und verbessere ihn in allen relevanten Aspekten: Performance, Lesbarkeit, Modularität, Skalierbarkeit und Einhaltung moderner Best Practices für die entsprechende Programmiersprache oder Technologie. Erhöhe die Lesbarkeit, indem du klare Kommentare hinzufügst, sprechende Namen für Variablen, Funktionen und Klassen verwendest und die Struktur optimierst. Entferne unnötige Wiederholungen, ineffiziente Algorithmen oder Redundanzen und stelle sicher, dass der Code robust, zukunftssicher und leicht erweiterbar bleibt.
Nutze semantisches HTML, moderne CSS-Techniken und effiziente JavaScript-Strukturen, falls relevant. Überprüfe, ob Frameworks oder Bibliotheken optimal integriert sind, und schlage sinnvolle Alternativen vor, falls nötig. Achte darauf, dass der Code responsiv und barrierefrei (Accessibility) ist.
Identifiziere und behebe potenzielle Fehler, Sicherheitslücken oder Schwächen. Beschreibe jede Änderung oder Verbesserung detailliert, dokumentiere größere Anpassungen mit kurzen Kommentaren und erkläre den Nutzen jeder Optimierung. Stelle sicher, dass der überarbeitete Code modernen Standards entspricht und optimal für zukünftige Entwicklungen geeignet ist.
```

## Website design

```
Bitte entwerfe ein komplett neues Design für diese Website, das modern, benutzerfreundlich und visuell ansprechend ist. Verwende dabei ausschließlich einen Darkmode als Basisdesign. Du kannst die Struktur und das Layout der Website vollständig verändern, um die Benutzerfreundlichkeit und die visuelle Wirkung zu maximieren.
Erstelle ein harmonisches Farbschema mit optimalem Kontrastverhältnis für Texte und Elemente, um eine gute Lesbarkeit zu gewährleisten. Nutze moderne Design-Prinzipien wie klare Typografie, konsistente Abstände, ansprechende visuelle Hierarchien und intuitive Navigation.
Das Design soll vollständig für mobile Geräte optimiert sein. Sorge dafür, dass das Layout sich an verschiedene Bildschirmgrößen anpasst. Implementiere bei Bedarf ein Hamburger-Menü oder andere flexible Navigationselemente, wenn der Bildschirmplatz zu klein wird.
Beschreibe das Design detailliert, einschließlich neuer Layouts, Schriftarten, Abständen und visueller Effekte. Überlege dir innovative Wege zur Präsentation von Inhalten und Navigation, die die Benutzererfahrung verbessern. Falls möglich, füge Vorschläge für Animationen oder Übergänge hinzu, die das Design lebendiger und dynamischer machen.
Das Design soll responsiv und barrierefrei (Accessibility) sein, sodass es auf allen Geräten und für alle Benutzergruppen gut funktioniert. Beschreibe abschließend die Änderungen und erkläre, wie diese die Funktionalität und Ästhetik der Website verbessern. Alle Kommentare im Code sollen auf Deutsch sein.
```
